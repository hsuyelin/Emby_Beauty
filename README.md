# 本项目是快速集成[Emby.CustomCssJS](https://github.com/Shurelol/Emby.CustomCssJS)的项目

# 使用说明

## step1: 下载最新版本
```bash
# 下载链接为示例链接，具体以最新版本为准
curl -o emby_beauty.tar.gz https://github.com/hsuyelin/Emby_Beauty/archive/refs/tags/0.2.0.tar.gz
```

## step2: 解压到当前文件夹
```bash
tar -xzf emby_beauty.tar.gz
```

## step3: 进入解压后的文件夹
```bash
# 0.2.0是示例版本号，具体以解压后的文件夹名称为准
cd Emby_Beauty-0.2.0/
```

## step4: 给脚本执行权限
```bash
chmod a+x beauty.sh
```

## step5: 执行脚本
```bash
./beauty.sh
```
